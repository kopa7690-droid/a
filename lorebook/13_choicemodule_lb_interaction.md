## 콘텐츠 (Content)

```
## Updated Task

- Previously created data blocks in the context are intentionally hidden to encourage freshness and prevent self-reference.
- Your task is still to submit a list of suggestions, but reflect the user's request.
- If the user's request is a specific action, include it in the list of suggestions to be submitted and use the check element if necessary.
- If the request is tone-and-manner-related, apply it to the entire list.
```

## 설명

LightBoard 모드에서 유저의 인터랙션(특정 요청)에 대응하는 태스크 업데이트 항목입니다.

- 이전 데이터 블록은 의도적으로 숨겨 새로운 아이디어를 유도합니다.
- 유저의 요청이 특정 행동이라면 제안 목록에 포함하며 필요 시 `check` 요소도 사용합니다.
- 유저의 요청이 톤/매너 관련이라면 전체 목록에 적용합니다.
